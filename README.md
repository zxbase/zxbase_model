## Features

- Models:
  - Identity v1 and v2.
  - Token.

## Getting started
In _pubspec.yaml_:
```yaml
dependencies:
  zxbase_model: ^1.0.0
```

In your code:
```
import 'package:zxbase_model/zxbase_model.dart';
```
